# Server Health Check Script

A PowerShell script that monitors Windows server health by checking disk space, memory usage, and CPU load.

## Features
- Displays disk space usage on C: drive
- Monitors RAM usage
- Checks CPU usage
- Alerts if thresholds are exceeded (disk <20%, memory >80%, CPU >80%)

## How to Use
1. Save the script to your machine
2. Open PowerShell as Administrator
3. Run: `.\health-check.ps1`
4. Review the output

## Requirements
- Windows PowerShell 5.0 or later
- Administrator privileges
- Windows operating system
